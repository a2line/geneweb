Module type Flatset.S

This module implements flat sets, which arel sets represented by sorted arrays.

The main goal of this implementation is to provide sets with an iterator interface that supports efficient seek operations.

Type of elements of the collection.

Build an flat set from a sequence of elements. The sequence is entirely forced while building the set.

to_seq t returns the sequence of elements of t in ascending ordering.

cardinal t returns the cardinal of the set.
