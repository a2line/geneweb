val key_after : (key -> int) -> 'a t -> key

key_after f_compare m browse map m to find the key k which gives f_compare k = 0. Raise Not_found if such key doesn't exists.

val next : key -> 'a t -> key

next k bt returns the smallest key that is bigger then k inside bt.
